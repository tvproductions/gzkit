# gz chores show

Display the CHORE.md content for one chore.

## Usage

```bash
gz chores show <slug>
```

## Description

Outputs the full CHORE.md specification for a single chore, including scope, acceptance criteria, and governance annotations.

## Exit Codes

| Code | Meaning |
|------|---------|
| 0 | Chore content displayed successfully |
| 1 | Chore not found or invalid input |
