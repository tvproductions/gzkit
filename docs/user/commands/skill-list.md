# gz skill list

List all discovered skills and their metadata.

## Usage

```bash
gz skill list [OPTIONS]
```

## Description

Displays all discovered skills in the repository, including name, version, status, and description. Skills are automatically discovered from `.claude/skills/`.

## Exit Codes

| Code | Meaning |
|------|---------|
| 0 | Skills listed successfully |
| 1 | Skill discovery failed |
