# OBPI-0.16.0-04 — template-engine

## ADR ITEM (Lite) — Level 1 WBS Reference

- Source ADR: `docs/design/adr/pre-release/ADR-0.16.0-cms-architecture-formalization/ADR-0.16.0-cms-architecture-formalization.md`
- OBPI Entry: `OBPI-0.16.0-04 — "Make gz agent sync vendor-aware; render canonical content to enabled vendor shapes"`

## OBJECTIVE (Lite)

Transform `gz agent sync control-surfaces` from a skill-mirroring tool into a proper
template engine that renders ALL canonical content (skills AND rules) into vendor-specific
shapes for ONLY enabled vendors. The engine reads `.gzkit/manifest.json` for vendor
enablement, reads canonical content from `.gzkit/skills/` and `.gzkit/rules/`, and
generates vendor surfaces with appropriate format transformations:

- Claude: `.claude/rules/*.md` with `paths:` YAML frontmatter
- Copilot: `.github/instructions/*.instructions.md` with `applyTo:` frontmatter
- Codex: `.agents/skills/` mirrors
- Gemini: `GEMINI.md` content sections
- OpenCode: `opencode.json` instruction references

## LANE (Lite)

Lite — ADR note + stdlib unittest + smoke (≤60s).

## ALLOWED PATHS (Lite)

- `src/gzkit/agent_sync.py` (major refactor — vendor-aware rendering)
- `src/gzkit/renderers/` (new — per-vendor rendering modules)
- `.claude/rules/` (generated output)
- `.claude/skills/` (generated output — already works)
- `.github/instructions/` (generated output — replaces hand-authored)
- `.github/skills/` (generated output — already works)
- `.agents/skills/` (generated output — already works)
- `tests/test_agent_sync.py`

## DENIED PATHS (Lite)

- `.gzkit/` canonical content (read-only for rendering)
- CI files, lockfiles

## REQUIREMENTS (FAIL-CLOSED — Lite)

1. `gz agent sync control-surfaces` reads `manifest.vendors` to determine which surfaces to generate
1. Disabled vendors produce no output (no stale mirrors left behind)
1. Enabled vendors get BOTH skill mirrors AND rule mirrors
1. Rule rendering transforms vendor-neutral frontmatter (`paths:`) into vendor-specific format (`applyTo:` for Copilot)
1. Generated files include a header comment: `<!-- Generated by gz agent sync — do not edit -->`
1. `gz agent sync --dry-run` shows what would be generated without writing
1. Existing skill mirroring behavior preserved exactly (regression proof)
